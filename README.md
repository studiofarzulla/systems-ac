# ASCRI

**Adversarial Systems & Complexity Research Initiative**

Live site: [systems.ac](https://systems.ac)

## About

ASCRI is a non-profit research initiative hosting papers on adversarial dynamics in governance, markets, cognition, and formal systems. Papers are organized by research programme and assigned working paper numbers (DAI/DP series).

## Tech Stack

- Static HTML/CSS/JS
- Node.js build pipeline
- Cloudflare Pages deployment

## Build

```bash
node build.js
```

Regenerates pages from `papers.json`.
