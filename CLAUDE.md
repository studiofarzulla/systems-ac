# CLAUDE.md

Instructions for Claude Code when working in this repository.

## Project Overview

**Site:** systems.ac
**Entity:** ASCRI (Adversarial Systems & Complexity Research Initiative)
**Purpose:** Academic face of ASCRI — research programmes, papers with Google Scholar metadata, theoretical framework.

**Architecture:** Static HTML + Node.js build script. Hosted on Cloudflare Pages. Auto-deploys from GitHub on push.

## Relationship to Other Sites

- **systems.ac** = the research initiative (theory, programmes, papers)
- **dissensus.ai** = the lab/org (team, services, manifesto) — links to systems.ac
- **farzulla.org** = personal academic profile (hosts PDFs)

## Design System

**Aesthetic:** Light academic — clean, authoritative, reading-focused. Think SFI, INET Oxford.

**Colors:**
- Background: `#FFFFFF` (white) / `#F8F7F5` (warm off-white)
- Accent: `#9B1B30` (muted crimson)
- Text: `#1A1A1A` (near-black)

**Typography:**
- Headings: Crimson Text (serif)
- Body: Inter (sans-serif)
- Code/metadata: JetBrains Mono

## Site Structure

```
systems-ac/
├── papers.json          # Paper metadata (source of truth)
├── build.js             # Generates paper pages, programme pages, sitemap, RSS
├── wrangler.json        # Cloudflare Pages config
├── public/              # Served by Cloudflare Pages
│   ├── index.html       # Hand-authored homepage
│   ├── framework.html   # Theoretical framework
│   ├── people.html      # Research team
│   ├── about.html       # About ASCRI
│   ├── contact.html     # Contact & collaboration
│   ├── css/ascri.css    # Single stylesheet
│   ├── assets/          # Favicons, logo
│   ├── papers/          # Generated by build.js
│   ├── programmes/      # Generated by build.js
│   ├── sitemap.xml      # Generated
│   └── feed.xml         # Generated
```

## Build Process

```bash
node build.js
# Generates papers/*, programmes/*, sitemap.xml, feed.xml
```

## Development

```bash
node build.js && python -m http.server 8000 --directory public
```

## Key Rules

- No PDFs in this repo — all PDF links point to farzulla.org/papers/
- Don't edit papers/*.html or programmes/*.html directly (generated)
- Rebuild after editing papers.json
- Publisher is "ASCRI" not "Farzulla Research"
